Overview

Docker is available as Community Edition (CE) for free and an Enterprise Edition (EE) subscription with software, support and certification. Docker is available for a wide range of operating systems for desktops, VMs, servers and cloud laaS -each optimized for and integrated with the target infrastructure. Subscription packages include an integrated Docker platform and tooling for container management and security.

Pricing Details
*Pricing plan varies by support tier selected during purchase
*Pricing is per node instance. An Instance is a single node deployment of Licensed Software or Supported Software installed on a physical or virtual computer or server, limited to two physical processor sockets in the case of a physical server. Every node within a EE managed cluster must be licensed (manager, registry, cache and worker nodes).
